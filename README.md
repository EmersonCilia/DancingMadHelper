  <strong>Helper plugin for Dancing Mad (Ultimate) P4.</strong>

## Features

* Tracks Dancing Mad debuffs.
* Displays whether each debuff is **Long** or **Short**.
* Lets you manually mark debuffs as **True** or **Fake**.


![Main Window](https://github.com/EmersonCilia/tentativeone/blob/master/Images/img.png?raw=true)

## Commands

* `/dmhelper` — Open the plugin window.
